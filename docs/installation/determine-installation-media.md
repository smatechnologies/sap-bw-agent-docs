# Determine installation media

## What is it?

The SAP BW Agent installation files can be delivered through several types of media. This page summarizes each option and where to run the setup program from.

In the documentation that follows, replace **\<media>** with your chosen method of delivery.

## Options

| Media | Where to run setup |
|---|---|
| **USB** | Plug the USB drive into the target machine, copy the installation files to a folder on the local hard drive, and run the installation from the hard drive. |
| **FTP** | Download the installation files directly to a folder on the target machine's hard drive, then run the installation from the hard drive. |
| **DVD-ROM** | Insert the DVD into the drive and run the installation from the DVD, or copy the files to the hard drive first. |
| **ISO** | Mount the ISO image as a virtual DVD and run the installation from that location. |

:::tip
Running the installer from the local hard drive (rather than over the network or off removable media) is the most reliable option.
:::
