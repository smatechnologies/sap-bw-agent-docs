# Silent mode

## What is it?

Silent mode is an unattended installation that runs the agent installer without an interactive setup wizard. Use it for scripted or repeatable deployments. The same silent installation procedure applies across OpCon agents.

For step-by-step instructions, refer to [Silent Mode](https://help.smatechnologies.com/opcon/core/installation/components#silent-mode) in the **OpCon Installation** online help.
